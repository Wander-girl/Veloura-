# 💎 Veloura - MERN Stack E-commerce Website

## 📌 Project Overview
Veloura is a full-stack e-commerce web application built using the MERN stack.  
It allows users to browse products by categories and add items to cart.  
An admin panel is also provided to add new products with image upload functionality.

---

## 🚀 Features

### 👤 User Side
- 🛍️ Browse products by categories:
  - Earrings
  - Rings
  - Necklaces
  - Hair Accessories
- 🛒 Add products to cart
- 📱 Responsive UI

### 👑 Admin Side
- ➕ Add new products
- 📤 Upload product images
- 📂 Category-based product management

---
